# Linked List Playlist Manager

A C++ playlist management program implemented with linked list data structures.

This project simulates a simple music playlist system and focuses on practicing pointer-based data structures, list operations, and object-oriented programming in C++.

## Features

* Add, remove, and manage songs in a playlist
* Traverse playlist data
* Organize playlist logic with custom C++ classes
* Practice linked list operations and pointer-based data structures

## How to Build

From the `linked-list-playlist` directory:

```bash id="nrepmc"
g++ src/main.cpp src/Playlist.cpp -I include -o playlist
```
