# Music Data Structures

A collection of C++ data structure projects for playlist management and music recommendation.

This repository includes small projects that explore linked lists, AVL trees, threaded AVL trees, and graph algorithms through music-related applications.

## Projects

| Project | Description | Main Concepts |
|---|---|---|
| [Linked List Playlist Manager](./linked-list-playlist) | A playlist management program implemented with linked list data structures. | Linked List, OOP |
| [AVL Playlist Manager](./avl-playlist-manager) | A playlist management program implemented with AVL Tree and Threaded AVL Tree. | AVL Tree, BST, Tree Traversal |
| [Music Recommendation Graph](./music-recommendation-graph) | A music recommendation system implemented with graph algorithms. | Graphs, BFS, DFS, Dijkstra |

## Technologies

- C++
- Object-Oriented Programming
- Data Structures
- Algorithms

## Purpose

This repository was created to practice implementing core data structures and algorithms in C++ through small application-based projects.
